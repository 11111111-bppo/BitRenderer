# BitRenderer

C++渲染器。

三方库：

[nothings/stb](https://github.com/nothings/stb)：读写图片。

[ocornut/imgui](https://github.com/ocornut/imgui)：GUI，后端使用Win32+DirectX 12。

[tinyobjloader/tinyobjloader](https://github.com/tinyobjloader/tinyobjloader)：加载obj模型
